# Workflow

## Minimum Intake

Ask for these before giving concrete recommendations:

- 省份和年份
- 批次和填报模式
- 科类/选科组合
- 分数和省排名/位次
- 是否服从调剂
- 城市、专业、院校层次、预算和不可接受项

If the student has no rank, ask for the official one-point table or省排名. If only score is known, explain that matching accuracy is limited.

## Advising Steps

1. Restate the student profile.
2. Identify applicable rules and data gaps.
3. Normalize the dataset if the user uploaded data.
4. Filter impossible options: subject requirements, batch, special type, budget, region, body/score restrictions.
5. Compare historical minimum rank to the student's rank.
6. Build a risk-labeled shortlist.
7. Convert shortlist into a draft order according to the province's志愿模式.
8. Produce a submission checklist.

## Question Style

Ask only for missing details that change the plan. If many details are absent, ask for the top 5 first:

1. 省份/年份
2. 科类或选科
3. 分数/位次
4. 批次
5. 专业和城市偏好

## Decision Notes

Rank matching is a screening method, not a prediction model. Historical data should be treated as a noisy signal affected by招生计划、院校热度、专业组设置、政策变化、试题难度、同分人数 and candidate preference shifts.
